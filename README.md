# Dark Hunter
Simple ray casting game

## Description
Game use ray casting method to make 3D space in 2D. In game is one testing map. Player has one gun for two rounds. Weapon has shooting and reloading animation. Shoot from gun make short flash effect. Sources of light in this game are lamps, withou tham is envirpoment dark. The game has sound effect for shooting/reloading and bg music.
